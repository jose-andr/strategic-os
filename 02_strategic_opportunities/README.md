# Strategic Opportunities

## Purpose

This folder contains opportunities that could increase strategic influence, support better decision-making, strengthen stakeholder relationships or create future career value.

Use this folder to capture, shape, review and track opportunities before they become active projects, career evidence or reusable knowledge.

## What belongs here

- New strategic opportunities
- Early opportunity notes
- Opportunity reviews
- Stakeholder-sponsored possibilities
- Stretch work options
- Future role or project possibilities
- Ideas that could create strategic influence
- Work that may connect customer and organisational outcomes
- Opportunities that need shaping before action

## What does not belong here

- Raw inbox notes that have not been triaged
- Full project documentation
- Completed decision briefs
- General career reflections
- Sensitive organisational source data
- Customer-level records
- Confidential stakeholder commentary
- Governed datasets

Use summaries, opportunity logic and safe-to-retain notes instead.

## Opportunity pipeline

Use these states to track an opportunity:

| State | Meaning |
|---|---|
| Captured | Noted as a possible opportunity |
| Shaping | Being clarified, framed or explored |
| Socialising | Being discussed with stakeholders |
| Active | Currently being pursued |
| Parked | Useful but not active now |
| Declined | Chosen not to pursue |
| Converted to project | Moved into project work |
| Converted to career evidence | Captured as career or promotion evidence |

## Opportunity assessment criteria

Review opportunities against:

- strategic value
- stakeholder value
- organisational value
- customer value
- career value
- evidence value
- timing
- effort
- trade-offs
- risk of distraction

## Suggested naming

Use date-first filenames.

Examples:

- `2026-06-15_opportunity-review_channel-strategy-advisory.md`
- `2026-06-15_opportunity_ai-enabled-service-design.md`
- `2026-06-15_opportunity_business-design-role.md`

## Review rhythm

Review this folder monthly.

Use the review to ask:

- What opportunities are emerging?
- Which opportunities need shaping?
- Which opportunities need stakeholder engagement?
- Which opportunities should become active?
- Which opportunities should be parked or declined?
- Which opportunities created career evidence?

## Review checklist

Before saving an opportunity, check:

- [ ] Is the opportunity clearly described?
- [ ] Is the strategic value visible?
- [ ] Is the next action clear?
- [ ] Is the opportunity safe to retain?
- [ ] Is sensitive information removed or generalised?
- [ ] Does this support the Strategic OS North Star?
- [ ] Should this be moved to a project, decision brief or career evidence folder?
