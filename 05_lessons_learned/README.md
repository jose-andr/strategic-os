# Lessons Learned

## Purpose

This folder contains reusable lessons from Strategic OS work, projects, decisions, experiments, stakeholder engagement and career development.

Use this folder to turn experience into patterns that can improve future decisions, delivery, influence and system design.

Lessons learned should help Strategic OS get smarter over time.

## What belongs here

- Lessons from completed work
- Lessons from decisions
- Lessons from stakeholder engagement
- Lessons from projects
- Lessons from analytics or data interpretation
- Lessons from AI-agent use
- Lessons from career development
- Retrospectives
- Mistakes and course corrections
- Reusable patterns discovered through practice

## What does not belong here

- Raw notes
- Full project documentation
- Sensitive organisational source data
- Customer-level records
- Confidential stakeholder commentary
- Governed datasets
- Private organisational records
- Lessons that are too vague to reuse

Use safe summaries and reusable patterns instead.

## Lesson types

Use these categories where useful:

| Lesson type | Use for |
|---|---|
| Decision lesson | What improved or weakened a decision |
| Stakeholder lesson | What helped or blocked alignment |
| Delivery lesson | What affected momentum, quality or follow-through |
| Analytics lesson | What clarified or confused evidence interpretation |
| Agent lesson | What improved or degraded agent output quality |
| Career lesson | What strengthened positioning, evidence or opportunity creation |
| System lesson | What improved Strategic OS itself |

Examples:

- `2026-06-15_lesson-learned_agent-continuity.md`
- `2026-06-15_lesson-learned_service-account-eofy-story.md`

## Review rhythm

Review this folder monthly and quarterly.

Use the review to ask:

- What patterns are repeating?
- What should change in the way Strategic OS is used?
- What should become a framework?
- What should become a template?
- What should inform agent rules?
- What should inform career evidence?
- What should be archived because it is no longer useful?

## Review checklist

Before saving a lesson, check:

- [ ] Is the lesson specific enough to reuse?
- [ ] Is evidence separated from interpretation?
- [ ] Is sensitive information removed or generalised?
- [ ] Does the lesson describe what should change next time?
- [ ] Could this become a framework, template or agent rule?
- [ ] Does this improve future decision-making?
- [ ] Is the lesson safe to retain in Strategic OS?
