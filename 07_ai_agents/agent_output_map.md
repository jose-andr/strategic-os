# Agent Output Map

## Purpose

This file maps each Strategic OS agent to its primary outputs, supporting templates and recommended storage locations.

Use this when deciding which agent to run, which template to use and where to save the result.

## Storage rules

### Save decision-ready advice in `03_decision_briefs/`

Use this when the output supports a recommendation, decision or strategic judgement.

### Save strategic opportunities in `02_strategic_opportunities/`

Use this when the output identifies or assesses a future opportunity.

### Save reusable learning in `05_lessons_learned/`

Use this when the output captures a pattern, mistake, improvement or repeatable insight.

### Save stakeholder patterns in `06_stakeholder_patterns/`

Use this when the output reveals a reusable stakeholder dynamic, influence pathway or engagement pattern.

### Save project-specific outputs in `08_projects/`

Use this when the output belongs to a specific project and is not yet reusable.

### Save career outputs in `01_career/`

Use this when the output supports development, positioning, promotion readiness or future role planning.

## Review before saving

Before saving any agent output, check:

- Is the output useful beyond the current moment?
- Is the recommended storage location clear?
- Is sensitive organisational source data removed?
- Are evidence, assumptions and interpretation separated?
- Is the next action clear?
- Should this become a reusable template, lesson or pattern?
