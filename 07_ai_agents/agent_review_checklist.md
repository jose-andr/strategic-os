# Agent Review Checklist

## Purpose

Use this checklist to review outputs created with Strategic OS agents before they are stored, shared or acted on.

The checklist helps keep agent outputs clear, useful, evidence-aware and aligned to the Strategic OS purpose.

## When to use

Use this checklist before:

- Sharing a decision brief
- Acting on a stakeholder alignment assessment
- Following a shipping recommendation
- Using career guidance
- Saving an opportunity review
- Turning an agent output into a project artefact
- Reusing an agent output in organisational work

## Core review questions

### 1. Decision clarity

- [ ] Is the decision, action or judgement clear?
- [ ] Is the output structured around what needs to happen next?
- [ ] Is the recommendation practical?
- [ ] Is the next step specific enough to act on?

### 2. Evidence quality

- [ ] Is evidence separated from interpretation?
- [ ] Are assumptions clearly marked?
- [ ] Are unsupported claims removed or caveated?
- [ ] Are source limitations visible?
- [ ] Is contextual evidence clearly distinguished from governed reporting?

### 3. Human judgement

- [ ] Is it clear where human judgement is required?
- [ ] Are sensitive interpretations checked before use?
- [ ] Are stakeholder assumptions treated carefully?
- [ ] Are governance or approval points visible?

### 4. Strategic alignment

- [ ] Does the output support the Strategic OS North Star?
- [ ] Does it help leaders make better strategic decisions?
- [ ] Does it connect customer outcomes and organisational outcomes where relevant?
- [ ] Does it strengthen the desired reputation?

### 5. Practical usefulness

- [ ] Is the output concise enough to use?
- [ ] Is the language clear and plain?
- [ ] Are trade-offs visible?
- [ ] Are risks and dependencies named?
- [ ] Is the recommendation proportionate to the problem?

### 6. Privacy and portability

- [ ] Does the output avoid storing organisational source data?
- [ ] Is sensitive information removed or generalised?
- [ ] Is the knowledge portable beyond one employer context?
- [ ] Is the file safe to keep in Strategic OS?

### 7. Reuse value

- [ ] Could this become a reusable pattern, template, lesson or case study?
- [ ] Should a lesson learned be created?
- [ ] Should the output be linked to an existing framework?
- [ ] Should a future prompt or agent rule be improved based on this?

## Agent-specific checks

### Sensemaking Agent

- [ ] Is the problem framed clearly?
- [ ] Are options compared?
- [ ] Are trade-offs visible?
- [ ] Is the recommendation decision-ready?

### Stakeholder Journey Agent

- [ ] Are stakeholders grouped accurately?
- [ ] Is resistance described fairly?
- [ ] Are influence pathways clear?
- [ ] Is the next conversation obvious?

### Shipping Coach

- [ ] Is the recommendation clearly Ship, Socialise, Refine or Stop?
- [ ] Is the minimum next action clear?
- [ ] Is the risk of waiting visible?
- [ ] Is the quality threshold realistic?

### Career Architect

- [ ] Does the output support long-term positioning?
- [ ] Does it create or strengthen career evidence?
- [ ] Are trade-offs visible?
- [ ] Is the recommendation grounded in real proof points?

## Final review decision

Choose one:

- [ ] Ready to use
- [ ] Needs light refinement
- [ ] Needs evidence check
- [ ] Needs human review
- [ ] Do not use yet
