# Career Guidance

## Purpose

Use this template to translate current work, feedback, opportunities and goals into practical career development guidance.

This template supports the Career Architect Agent.

## When to use

Use this when:

- Reviewing career direction
- Preparing for promotion
- Assessing capability gaps
- Translating project work into career evidence
- Deciding where to focus effort
- Shaping a clearer strategic career narrative
