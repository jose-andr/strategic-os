# Promotion Readiness Assessment

## Purpose

Use this template to assess how current work, achievements, evidence and capability development support progression toward a future role, level or promotion.

It helps the user clarify:

* target role or progression goal
* current readiness
* strongest evidence
* evidence gaps
* capability gaps
* promotion narrative
* development priorities
* stakeholder or sponsor input needed
* next practical action

This template supports the Career Architect Agent.

North Star:

> Increase strategic opportunities by helping leaders make better strategic decisions.

## When To Use

Use this template when:

* preparing for a promotion conversation
* reviewing readiness against a target role
* translating project work into evidence
* identifying gaps in experience or proof points
* preparing application examples
* reviewing progress over time
* deciding whether to seek a stretch opportunity
* planning a development conversation
* shaping a promotion narrative
* assessing whether evidence is strong enough to use

Do not use this template to inflate achievements, invent impact, or turn confidential organisational context into public career claims.

## Inputs

Useful inputs may include:

* target role description
* selection criteria

* performance goals

## Review Checklist

Before using this assessment, check:

* Is the target role or progression goal clear?
* Is evidence separated from interpretation?
* Are claims supported by examples?
* Are evidence gaps specific and actionable?
* Does the narrative support the desired reputation?
* Are claims grounded and proportionate?
* Is sensitive information removed or generalised?
* Is human review required before use?
* Is there a clear next action?
* Could this assessment support a real promotion conversation?
* Does it support long-term strategic opportunity?

## Output Quality Bar

A useful promotion readiness assessment should be:

* evidence-based
* proportionate
* specific about capability
* clear about gaps
* practical about development actions
* useful for a real conversation
* careful with confidential context
* authentic to the user’s actual work
* connected to future strategic opportunity

## Failure Modes

This template is not working if:

* it turns activity into inflated achievement
* it invents impact
* it ignores evidence gaps
* it uses generic promotion language
* it hides development needs
* it stores sensitive information unnecessarily
* it makes unsupported claims about organisational value
* it gives a readiness judgement without evidence
* it creates a narrative the user cannot credibly defend
