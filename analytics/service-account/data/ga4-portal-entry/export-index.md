# GA4 Portal Entry Export Index

Use this file to track refreshed GA4 CSV exports stored in this folder.

GA4 exports should be aggregated, non-sensitive, and approved for repository storage before being added here.

## Export register

| File | Reporting window | Export date | Analysis area | Metadata file | Approved for use? | Notes |
|---|---|---|---|---|---|---|
|  |  |  |  |  | Yes / No |  |

## Analysis areas

Use one of the following analysis areas:

- portal entries
- device mix
- entry pages
- weekday pattern
- hour-of-day pattern
- login vs sign-up split
- source channel
- business-hours share

## Status values

Use:

- `Draft`
- `Pending validation`
- `Approved for supporting report`
- `Approved for slide`
- `Do not use`

## Usage rule

Only use GA4 export values in the EOFY celebration slide or supporting report when:

- the CSV is listed in this index
- the matching metadata file is completed
- the reporting window is confirmed
- the export is approved for use
- caveats are documented
